О проекте "Информативный Столин"

Проект создан для гостей города Столина. С помощью бота или этого сайта можно находить информацию о Столине. Мы создали карты, котрые помогут вам найти необходимые места. В будущем проект будет переделан под весь Столинский район, потому что в самом большом районе Беларуси есть на что посмотреть.

Telegram-бот

Для этого проекта был создан telegram-бот. В нем можно найти всю инфрормацию о Столине

Сайт проекта "Информативный Столин"

Сайт создан специально чтобы узнавать информацию про Столин с помощью интернет-браузера.

Сервисы

Сервисы, которые помогли созадать проект

Robochat.io- это конструктор чат-ботов для социальной сети ВКонтакте и Telegram

Конструктор карт Яндекс- позволяет создать карту онлайн без навыков программирования. Можно отметить на карте нужные объекты, нарисовать схему проезда, маршрут прогулки или отметить зону доставки.

Telegram- кроссплатформенная система мгновенного обмена сообщениями

Github pages- позволяет загружать репозитории кода для хранения в системе управления версиями Git.
